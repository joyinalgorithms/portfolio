Project LIF - AI Lipreading System

AI/ML Ongoing

Project Overview

An innovative AI-powered lipreading system that uses 3D CNNs and BiLSTMs to classify Filipino syllables from video footage of mouth movements. This cutting-edge research project aims to improve accessibility for the hearing-impaired community.

Key Features

  • 3D Convolutional Neural Networks for spatial-temporal analysis
  • BiLSTM networks for sequence modeling
  • Filipino syllable classification system
  • Real-time video processing capabilities
  • Advanced computer vision techniques
  • Accessibility-focused design

Technology Stack

Python TensorFlow OpenCV NumPy Pandas Jupyter Google Colab
